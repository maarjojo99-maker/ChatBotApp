# ChatBot App

 A modern AI Chatbot interface featuring a sidebar for chat history and a main chat window.

## Features
- Sidebar for managing recent chats.
- Interactive chat interface.
- New chat creation and chat switching.
- Responsive design with CSS Modules.

## Key Components
- `Sidebar`: Handles navigation and chat history.
- `ChatBot`: The main messaging interface.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
